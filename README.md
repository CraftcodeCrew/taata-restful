# taata
REST-Service

Base URL: http://localhost:8080/api

Category resource:

| URI              | GET                | POST               | PUT               | DELETE            |
|------------------|--------------------|--------------------|-------------------|-------------------|
| /categories      | List of categories | Create a category  | Not allowed       | Not allowed       |
| /categories/{id} | A category         | Not allowed        | Update a category | Delete a category |
